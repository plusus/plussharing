# plussharing
Nautilus extension for file sharing in the PLUS distribution

## Dependencies
### Debian packages
#### Jessie

* python-nautilus
* libc6
* libssl1.0.0
* libffi6
* libffi-dev
* libssl-dev
* python3-pyasn1
* python3-pyasn1-modules
* python3-six
* python3-pycparser
* python3-pyside
* python3-mysql.connector
* python3-apt

#### Jessie-backports

* python3-setuptools
* python3-pkg-resources
* python3-idna

### Python packages (provided as dist-packages)

* cffi
* cryptography
* paramiko
* pysftp
